## Problem

Prompt engineering is arguably the highest-leverage skill of the decade and almost nobody is teaching it well. The existing material is either six-hour YouTube lectures or glorified cheat sheets. Neither builds intuition. Neither makes you come back tomorrow.

Duolingo solved this for languages using behaviour design, not content volume. Prompt engineering deserves the same treatment.

## Approach

Prompto is a mobile-first daily training app. The whole shell is React 18 + Vite 5 + strict TypeScript — no CSS framework, no router dependency, no backend. Progress lives entirely in `localStorage`.

The loop is deliberately small:

- **Daily mission** — five minutes, one concept, one measurable attempt.
- **Instant scoring** — the mission ships with a structured rubric the app runs client-side.
- **Retry** — you get the same scaffold back with diffable feedback on what moved the score.
- **Streak** — visible progress so tomorrow costs less willpower than today.

Hash-based routing keeps the bundle tiny; Vitest keeps the core scoring logic deterministic.

## Outcome

Public beta at [prompto-iota.vercel.app](https://prompto-iota.vercel.app). The repo is now scaffolded with Cavekit — 9 domain kits (5 reverse-engineered from Loop 1, 4 drafted for Loop 2: AI scoring seam, mission history, streak calendar, CI) and a 36-task build site with full acceptance-criteria coverage. Next: freemium monetization — free daily missions, Pro unlocks red-team mode and "Explain My Error."
