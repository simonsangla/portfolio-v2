## Problem

Prompt engineering is arguably the highest-leverage skill of the decade and almost nobody is teaching it well. Existing material is either six-hour YouTube lectures or glorified cheat sheets. Neither builds intuition. Neither makes you come back tomorrow.

Duolingo solved this for languages using behaviour design, not content volume. Prompt engineering deserves the same treatment.

## Approach

Dual Pronto is a mobile-first daily training app. The shell is React 18 + Vite 5 + strict TypeScript — no CSS framework, no router dependency, no backend. Progress lives entirely in `localStorage`.

The mission format is deliberately small:

- **Daily mission** — five minutes, one concept, one measurable attempt.
- **Instant scoring** — each mission ships with a structured rubric the app runs client-side.
- **Retry** — the same scaffold comes back with diffable feedback on what moved the score.
- **Streak** — visible progress so tomorrow costs less willpower than today.

Hash-based routing keeps the bundle tiny; Vitest keeps the core scoring logic deterministic.

## Outcome

Public beta at [prompto-iota.vercel.app](https://prompto-iota.vercel.app). The repo is scaffolded with Cavekit — 9 domain kits (5 reverse-engineered from the first build, 4 drafted for the next: AI scoring seam, mission history, streak calendar, CI) and a 36-task build site with full acceptance-criteria coverage. Next: freemium tier — free daily missions, Pro unlocks red-team mode and "Explain My Error."
