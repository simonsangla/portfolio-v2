Different problem sizes, different audiences.

**MetricPilot** is the big one — the Revenue Root Cause Engine productized. It runs on your Snowflake, decomposes a KPI into its additive drivers, detects change-points, and surfaces a ranked list of explanations. This is what the fixed-scope engagement delivers.

**Dual Pronto** is a daily-training web app for practical prompt work. Self-paced, low stakes, designed for operators who want to get sharper with LLMs without sitting through a course.

**Prompt Optimizer** takes a rough prompt and returns a scored, structured one. Local-only, bring-your-own Gemini key. Useful as a daily driver for anyone writing prompts for work.

**Theme Forge** is a visual theme editor for design tokens — adjust live, export to six formats. Tool for design engineers.

All four ship from the same agentic build workflow; only the first is a paid engagement.
